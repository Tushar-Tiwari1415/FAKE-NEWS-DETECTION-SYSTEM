# 📰 Fake News Detection System

A Machine Learning-based web application that detects whether a given news article is **real or fake**. The system uses Natural Language Processing (NLP) techniques and a trained classification model to analyze news content and make predictions.

---

##  Features

- Detects fake vs real news
- Text preprocessing using NLP techniques
- Trained ML model for classification
- Fast prediction results

---

##  Tech Stack

- Python
- Scikit-learn
- Pandas
- NumPy
- NLP (TF-IDF / Vectorizer)
- BERT
- SVM

---
## Working

User enters a news article
Text is cleaned and preprocessed
Feature extraction using TF-IDF / CountVectorizer
Trained ML model predicts:
🟢 Real News
🔴 Fake News

---

## Model Used
Logistic Regression
SVM
BERT
TF-IDF Vectorizer for feature extraction
Trained on labeled dataset of fake and real news
